### **Conditional Statements
- **1 if**
 ```c
 if (condition)
 {
 //code
 }
``` 
- **2. if-else**
```c
if (condition)
{
//code
}
else (condition)
{
//code for else
}
```
- **3. if-else ladder**
```c
if (condition 1)
{
//code 
}
else if (condition 2)
{
//code if condition1 false and condition 2 true
}
else {
//code for else
}
```
- **4. switch**
```c
switch (variable)
{
case 1:/* code when  variable==1*/ break;
case 2 : /*code */ break;
default : 
}
```

### **Loop**
- **1. while**
```c
while(condition)

//code
}
```
- **2. do-while**
```c
do
{
//code
}
while();
```
- **3. for loop**
```c
for (intilization ; condition; increment/decrement)
{
//code
}

//NOTE : we can write multiple intilization and inc/dec statements 
```
